# Student Performance Analysis

Exploratory data analysis of student performance factors using Python (NumPy, Pandas, Matplotlib, Seaborn).

## Dataset

`data/StudentPerformanceFactors.csv` — 6,607 student records with 20 features including study habits, attendance, family background, and exam scores.

## Figures

All generated charts are in `figures/`:

| File | Content |
|------|---------|
| `fig1_score_overview.png` | Score distribution, pass/fail rate, gender comparison |
| `fig2_study_habits.png` | Hours studied / attendance / sleep vs exam score |
| `fig3_family_factors.png` | Parental involvement, family income, internet access |
| `fig4_motivation_school.png` | Motivation level, teacher quality, peer influence |
| `fig5_correlation_heatmap.png` | Correlation matrix of numeric features |
| `fig6_attendance_analysis.png` | Deep dive into attendance vs performance |

## Usage

```bash
pip install -r requirements.txt
python student_analysis_beginner.py
```
